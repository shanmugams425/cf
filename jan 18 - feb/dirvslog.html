Diffrence between "console.log" and "console.dir".

  1. console.dir method is used mainly to display properties of an object. It outputs the list of properties of an object in a readable and interactive manner.

  2. 'dir' in the console.dir stands for the directory which represents the listing of elements or objects properties.

  3. Use console.log to display the output of anything in the console including strings, numbers, objects, etc. Use console.dir if you want to display the properties of an object in a hierarchical manner in JSON like format.
